# Vertical Gauge Font

`VerticalGauge-Regular.ttf` is used to render the vertical gauge progress style shown in the showcase.

If you need this gauge style in an actual project, merge `VerticalGauge-Regular.ttf` with the font you are using. FontForge can be used to merge the fonts.

The Unicode layout is shown below. The glyphs are placed in the Private Use Area:

| Code point | Description   |
| ---------- | ------------- |
| `U+E400`   | gauge level 0 |
| `U+E401`   | gauge level 1 |
| `U+E402`   | gauge level 2 |
| `U+E403`   | gauge level 3 |
| `U+E404`   | gauge level 4 |
| `U+E405`   | gauge level 5 |
| `U+E406`   | gauge level 6 |
| `U+E407`   | gauge level 7 |
| `U+E408`   | gauge level 8 |
